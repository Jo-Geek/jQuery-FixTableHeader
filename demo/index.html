jQuery-fixTableHeader

A jQuery plugin to fix header or footer in a table

Jones Vinoth Joseph

Developer - .Net(C#|VB), HTML, CSS, jQuery
Fixed header Fixed footer Header and footer Multiple Header Without grouping

Fixed header (This is the default mode, keeps header row fixed)

Example on the right shows a demo of a single row fixed as header.
